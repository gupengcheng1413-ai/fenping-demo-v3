## ✅ 最终解决方案

### 关键改变

**1. 移除边距限制**
```typescript
// 修改前
const scaleX = (windowWidth - 16) / VIEW_W;
const scaleY = (windowHeight - 16) / VIEW_H;

// 修改后
const scaleX = windowWidth / VIEW_W;
const scaleY = windowHeight / VIEW_H;
```

**2. 移除最大缩放限制**
```typescript
// 修改前
const newScale = Math.min(scaleX, scaleY, 1); // 最大不超过 1

// 修改后
const newScale = Math.min(scaleX, scaleY); // 无限制
```

### 工作原理

**竖屏模式：**
- 屏幕宽度 < 屏幕高度
- 按宽度缩放：`scale = screenWidth / 1640`
- 内容宽度 = 屏幕宽度（100%）
- 内容高度 = 348 × scale

**横屏模式：**
- 屏幕宽度 > 屏幕高度
- 按高度缩放：`scale = screenHeight / 348`
- 内容高度 = 屏幕高度（100%）
- 内容宽度 = 1640 × scale

## 🔍 技术细节

### 完整的缩放逻辑

```typescript
const [scale, setScale] = useState(1);

useEffect(() => {
  const updateScale = () => {
    const windowWidth = window.innerWidth;
    const windowHeight = window.innerHeight;

    // 计算两个方向的缩放比例
    const scaleX = windowWidth / VIEW_W;   // 宽度缩放
    const scaleY = windowHeight / VIEW_H;  // 高度缩放
    
    // 选择较小的缩放比例，确保内容完全显示
    const newScale = Math.min(scaleX, scaleY);

    setScale(newScale);
  };

  updateScale();
  window.addEventListener('resize', updateScale);
  return () => window.removeEventListener('resize', updateScale);
}, []);
```

### CSS Transform

```typescript
<div
  style={{
    width: VIEW_W,
    height: VIEW_H,
    transform: `scale(${scale})`,
    transformOrigin: 'center center'
  }}
>
```
