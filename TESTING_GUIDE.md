### 4. 分屏滚动测试（核心功能）

这是应用最复杂的交互功能，必须重点测试。

**测试步骤：**
1. 从历史页面进入"英文句子-分屏"模块
2. 页面应该分为左右两栏：
   - 左侧：英文句子
   - 右侧：中文翻译
   - 中间：1px 分割线

**测试场景 A：左侧滚动同步**
1. 用鼠标滚轮在左侧区域滚动
2. 右侧应该自动滚动到对应的中文句子
3. 当前活跃的句子应该高亮为黄色（#ffd34d）
4. 第一句始终为橙色（#ec9709）

**测试场景 B：右侧滚动同步**
1. 用鼠标滚轮在右侧区域滚动
2. 左侧应该自动滚动到对应的英文句子
3. 同样的高亮规则

**测试场景 C：锁定机制**
1. 在左侧按住鼠标左键（不松开）
2. 此时滚动左侧，右侧应该同步
3. 但如果右侧有滚动事件，不应该影响左侧
4. 松开鼠标后，恢复正常双向同步

**预期行为：**
- ✅ 滚动流畅，无卡顿
- ✅ 同步精确，对应句子正确
- ✅ 高亮颜色正确
- ✅ 无递归滚动（不会出现抖动）

## 故障排查

### 问题：页面显示空白

**可能原因：**
- 开发服务器未启动
- 浏览器缓存问题
- JavaScript 错误

**解决方案：**
1. 检查开发服务器是否运行：`curl http://localhost:5173/`
2. 清除浏览器缓存：Ctrl+Shift+Delete
3. 检查控制台是否有错误信息

### 问题：图片显示 404

**可能原因：**
- `figma:asset/` 路径解析失败
- 图片文件未复制

**解决方案：**
1. 检查 `vite.config.ts` 是否存在
2. 检查 `src/assets/` 目录是否包含图片
3. 重启开发服务器

### 问题：样式错乱

**可能原因：**
- Tailwind CSS 未正确加载
- 主题文件缺失

**解决方案：**
1. 检查 `default_shadcn_theme.css` 是否存在
2. 检查 `postcss.config.mjs` 是否存在
3. 清除 Vite 缓存：`rm -rf node_modules/.vite`
4. 重启开发服务器

### 问题：分屏滚动不同步

**可能原因：**
- JavaScript 错误
- 状态管理问题

**解决方案：**
1. 检查控制台是否有错误
2. 刷新页面重试
3. 检查 `src/app/components/ensentence-fenping.tsx` 是否正确复制
